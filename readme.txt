== Description ==
Enables Gplus crawling for Ushahidi copy

=== Requires ===
UshahidiSocialMedia plugin (https://github.com/ditorelo/Ushahidi-SocialMedia)

== Installation ==
1. Create a directory called "socialmedia_gplus" inside your /plugins/ directory
2. Copy the entire all files directory into your /plugins/socialmedia_gplus directory.
3. Activate the plugin.
4. Go to Settings > SocialMedia > G+ to enter your API credentials.
